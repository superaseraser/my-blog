自动化建站：一键更新脚本

为了提高更新效率，我编写了一个 Windows 批处理脚本（.bat）。只需双击运行，即可自动完成代码暂存、提交并推送到 GitHub 仓库。

### 📄 脚本代码 (update.bat)

```batch [update.bat]
@echo off
:: 设置编码为 UTF-8，防止中文乱码
chcp 65001 >nul

title 正在更新我的学术主页...
echo ---------------------------------------
echo 🚀 开始同步代码到 GitHub...
echo ---------------------------------------

:: 1. 暂存所有更改
git add .

:: 2. 自动提交，备注包含当前日期和时间
set msg=Update at %date% %time%
git commit -m "%msg%"

:: 3. 推送到云端 (主分支为 master)
echo 📤 正在上传，请稍候...
git push origin master

echo ---------------------------------------
echo ✅ 恭喜！代码已成功推送到 GitHub。
echo 🌐 Vercel 正在后台自动构建新版本...
echo ---------------------------------------

pause
